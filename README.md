# About

GCP 上で GCE を 3 台建てて簡素な (insecure な) Kubernetes 構築ハンズオンをやるためのリポジトリ

* 動作確認環境
    * Ubuntu 20.04
    * Kubernetes v1.19.7
    * Etcd v3.4.14
    * Flannel v0.11.0

## ./docs

Kubernetes構築用ガイド

## ./manifests

Kubernetes構築時の検証用manifestファイル群

## ./scripts

GCP上にハンズオン環境を構築するシェルスクリプト & Ansible Playbook
